Welcome to the homepage of Codesearch, an application that helps you keep an overview of your source code.
Codesearch gives you the power to search and view all code that is managed by version control systems such as Subversion, Git or Bazaar.
You can easily extend Codesearch with your own plugins, currently for these features:
  • Version control systems
  • Syntax highlighting
  • Code analyzing(Abstract Syntax Tree parsing)
