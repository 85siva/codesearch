Install instructions

Prerequisites:

  • Java 6 or higher
  • Apache Maven build tool
  • Apache Tomcat 6 or higher
  • MySQL Database(for code analyzing support)

Basic setup

  1. Make sure you have everything listed above.
  2. Get the source code.
  3. Switch to the src directory
  4. Edit 
    commons/src/main/resources/codesearch_config.xml
    (better solution in development))
  5. Execute 
    mvn clean package -DskipTests=true
  6. Copy 
    src/indexer/target/indexer-{version}.war
src/searcher/target/searcher-{version}.war
    to your Tomcat webapps directory.
  7. Run your Tomcat
  8. Go to {tomcat-address}/searcher-{version}

Code Analyzing

If you want code analyzing(outline, navigation between method calls and definitions etc.), you have to do the following:
  1. enable code analyzing in the config(see above))
  2. configure the database connection in the config
  3. create the database user and execute the SQL script in 
    resources/db-script.sql
  4. make sure the configured user has access to the created db(codesearch_db)
  5. deploy CodeAnalyzingPlugins on your Tomcat server
